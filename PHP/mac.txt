Switching PHP Versions
https://stackoverflow.com/questions/34909101/how-can-i-easily-switch-between-php-versions-on-mac-osx
1) Check installed php versions by brew, for sure everything good
> brew list | grep php
2) unlink the php version you want to unlink
brew unlink php@7.4
3) link the php version you want to link install if needed
brew install php@7.4
brew link php@7.4
